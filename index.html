Examining Judicial Racial Bias

Investigated fairness in binary classification by analyzing the impact of demographic features on model predictions. A logistic regression model was trained to predict risk outcomes, and the influence of each group was assessed through the learned coefficients.

The chart shows the model’s coefficient values for each demographic category. A significantly higher coefficient for one group indicates potential bias, suggesting the model may associate that group with a higher likelihood of the predicted outcome. This analysis highlights the importance of auditing models for fairness and accountability.

The chart shows Alpha-Beta's performance over 100 games per depth level, highlighting its ability to improve with deeper lookahead.

Positive EV Sports Betting

Developed a value-betting model that compares live odds from local sportsbooks to sharp sportsbook lines to identify +EV (positive expected value) opportunities. The system uses real-time data from TheOddsAPI and calculates optimal bet sizing using the Kelly Criterion to maximize long-term profit. All bets are logged in a database and can be updated with a simple web interface for viewing current and past bets.

Predictive Modeling & Data Analysis on Housing Dataset

Built a regression model to predict log-transformed home sale prices using housing features like square footage, location, and condition. The project involved data preprocessing, feature selection, and model tuning using validation data to ensure accuracy and generalization.

The chart shows residuals from one of the models, revealing patterns of model bias—underpredicting high-value homes and overpredicting low-value ones. These insights helped refine the model and improve predictive performance.
