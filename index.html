Positive EV Sports Betting

Developed a value-betting model that compares live odds from local sportsbooks to sharp sportsbook lines to identify +EV (positive expected value) opportunities. The system uses real-time data from TheOddsAPI and calculates optimal bet sizing using the Kelly Criterion to maximize long-term profit. All bets are logged in a database and can be updated with a simple web interface for viewing current and past bets.
